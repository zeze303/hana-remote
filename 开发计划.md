# hana-remote 开发计划

> 通过 Web 浏览器远程使用 Hanako。
> 架构：浏览器 → Render 中继 ←outbound WSS← 工作电脑 Hanako 插件
> 仓库：https://github.com/zeze303/hana-remote

---

## 项目概况

| 字段 | 内容 |
|------|------|
| 前端风格 | 极简黑，原生 JS + CSS |
| 编辑器 | Monaco Editor（CDN 引入） |
| 认证 | 密码登录 → JWT → WSS |
| 文件范围 | Windows 全盘访问，文本编辑，写入需切模式 |
| 聊天 | 与 Hanako 对话，流式渲染回复 |
| 部署 | Relay 部署 Render，绑定域名，HTTPS |
| Git | 改完代码自动 commit + push |

## 开发约定

1. **每次开发前**：自动读取本文件，查看当前状态
2. **commit message 格式**：`<type>: <中文说明>`
   - `feat:` 新功能
   - `fix:` 修复
   - `chore:` 杂项（配置、初始化等）
   - `refactor:` 重构
   - `docs:` 文档
3. **每次 commit 后自动 push**
4. **完成一个子任务后**，将 `[ ]` 改为 `[x]` 并提交更新
